# Sauce_Demo Automation Project

## Project Overview
This project automates the Sauce Demo website: [https://www.saucedemo.com](https://www.saucedemo.com)  
It includes end-to-end testing for:

- Login functionality
- Adding items to the cart
- Checkout: Your Information
- Checkout: Overview
- Checkout Complete

## Technologies Used
- Java
- Selenium WebDriver
- TestNG
- Maven
- Apache POI (Excel Utilities)
